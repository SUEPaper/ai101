# 机器学习

机器学习课程关注经典算法、模型评估和问题建模，目标是让学习者理解模型为什么有效、什么时候失效，以及如何选择合适的建模方法。

## 课程目标

- 理解监督学习、无监督学习和模型泛化的基本概念。
- 掌握常见分类、回归和聚类算法的使用方式。
- 能够使用交叉验证、评价指标和误差分析判断模型质量。
- 建立特征工程、模型选择和调参的基本方法。

## 核心内容

- 机器学习问题定义
- 线性模型、树模型与集成学习
- 分类、回归与聚类任务
- 模型评估与交叉验证
- 特征工程与模型调参

## 学习产出

完成课程后，学习者应能针对结构化数据任务建立基线模型，比较多个算法方案，并用明确指标说明最终模型选择。
