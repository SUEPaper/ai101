# 数据科学

数据科学课程面向 AI 学习的前置能力建设，目标是让学习者能够从原始数据出发，完成数据理解、清洗、分析和表达。

## 课程目标

- 理解数据类型、数据质量和数据分析流程。
- 掌握数据清洗、缺失值处理、异常值识别和特征整理。
- 使用统计指标和可视化方法发现数据规律。
- 能够产出结构清晰、结论可信的数据分析报告。

## 核心内容

- 数据科学工作流
- Python 数据处理基础
- 描述性统计与探索性数据分析
- 数据可视化与结果表达
- 面向机器学习的数据准备

## 学习产出

完成课程后，学习者应能独立分析一个中等规模数据集，并形成包含数据背景、处理过程、关键发现和后续建模建议的分析文档。
